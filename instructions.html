Running the model locally

You may wish to run the model locally by downloading all the HTML and javascript files. In order to do this, you need to set up the browser to allow local file access. Note that this potentially allows malicious websites to access local files, so please only run the model (and don't visit other websites) while using this)

Press Windows key, type "run"

Enter C:\Program Files (x86)\Google\Chrome\Application\chrome.exe --allow-file-access-from-files
